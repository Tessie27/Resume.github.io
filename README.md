# Le Tezz Khan - Portfolio

Interactive portfolio showcasing my skills and experience as a Junior Automation Engineer. Built with modern web technologies and accessibility in mind.

## 🛠️ Tech Stack

- **Framework:** Next.js 14
- **Styling:** CSS3 with CSS Variables
- **Icons:** Font Awesome
- **Deployment:** Vercel
- **Animation:** CSS Keyframes

- ✅ Responsive Design
- ✅ Animated Code Background
- ✅ Smooth Scrolling Navigation
- ✅ Interactive Easter Egg
- ✅ Accessible Components
- ✅ Mobile-Friendly
